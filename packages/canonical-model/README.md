# Canonical Model

The Canonical Model defines the shared domain objects used throughout the Numeria platform.

Every application depends on this package.

The model is intentionally technology-independent.

It represents educational concepts rather than database structures.

## Core Objects

- Entity
- Relationship
- Character
- Story
- Lesson
- Artifact
- Location
- Organization

## Goals

- One shared model
- Canon-first
- Versioned
- AI-friendly
- Knowledge Graph ready
