# Numeria Forge

> Forge mathematical worlds from canonical knowledge.

Numeria Forge is the command-line creation engine for the Numeria Foundation.

## Features

- Create canonical concepts
- Create characters
- Create stories
- Generate lessons
- Generate animations
- Generate short videos
- Validate the repository
- Build the learning graph

## Initial Commands

```bash
forge --help
forge version
forge init
```

## Development Installation

```bash
python -m pip install -e ".[dev]"
```
