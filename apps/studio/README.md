# Numeria Studio

## Version

v0.1.0 — Genesis

## Mission

Numeria Studio is the official authoring and exploration environment for
the Numeria Foundation.

It provides:

- Canon Explorer
- Graph Visualization
- Story Composer
- Curriculum Builder
- AI Tutor
- Canon Validator

The canonical source of truth remains the YAML knowledge graph stored in
the Numeria Foundation repository.
