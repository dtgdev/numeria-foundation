**Relationship ID:** NUM-REL-000003
**Status:** CANON

## Relationship

The Silver Delta is Derivative's signature artifact.

It alerts him when meaningful change is nearby, but it never tells him
the answer.

Derivative must still investigate, compare evidence, and reason through
the mystery.
