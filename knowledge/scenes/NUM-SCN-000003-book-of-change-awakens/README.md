# The Book of Change Awakens

**Scene ID:** NUM-SCN-000003  
**Status:** CANON

## Purpose

The Silver Delta awakens the Book of Change, but the Book does not reveal
an answer automatically.

Derivative must first observe something that changed and explain the
evidence.

## Educational Meaning

This scene teaches that:

- observation comes before explanation
- evidence matters
- magical tools do not replace reasoning
- discoveries must be understood before they are recorded

## Canon Rule

The Book of Change never records guesses as truth.
