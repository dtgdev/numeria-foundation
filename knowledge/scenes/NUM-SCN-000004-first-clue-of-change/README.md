# The First Clue of Change

**Scene ID:** NUM-SCN-000004  
**Status:** CANON

## Purpose

Derivative discovers the first real clue recorded by the Book of Change.

A sunflower has grown taller.

Delta does not explain the answer. She asks Derivative to compare the
sunflower's earlier height with its current height.

## Educational Meaning

This scene introduces:

- before-and-after comparison
- observable growth
- changing values
- evidence-based reasoning

## Canon Rule

The Book of Change responds only after Derivative describes the change
using evidence.
