# The Pattern Appears

**Scene ID:** NUM-SCN-000006  
**Status:** CANON

## Purpose

Derivative studies several sunflower measurements instead of relying on
one before-and-after comparison.

He notices that the height changes according to a pattern.

Delta guides him to ask:

> What rule connects the measurements?

## Educational Meaning

This scene introduces:

- repeated observation
- patterns across time
- changing values
- fixed measurement intervals
- the beginning of functional relationships

## Canon Rule

A pattern must be supported by multiple observations.

One measurement may reveal a difference.

Several measurements may reveal a rule.
