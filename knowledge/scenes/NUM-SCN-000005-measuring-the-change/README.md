# Measuring the Change

**Scene ID:** NUM-SCN-000005  
**Status:** CANON

## Purpose

Derivative moves from noticing change to measuring it.

He compares the sunflower's earlier height with its current height and
explains the difference using evidence.

Only then does the Book of Change reveal its first written page.

## Educational Meaning

This scene introduces:

- measuring a difference
- changing and fixed values
- before-and-after evidence
- recording a mathematical observation

## Canon Rule

The Book of Change records only discoveries that have been observed,
compared, and explained.
