# Function

**Character ID:** NUM-CHR-000003  
**Title:** Builder of Paths  
**Status:** CANON  
**Role:** Guardian of Relationships

> Give me an input, and I will show you where it leads.

## Overview

Function is the builder of dependable connections in Numeria.

Function creates paths that follow clear rules. When the same valid input
enters a path, it always reaches the same output.

Function helps others understand how one quantity, choice, or event relates
to another.

## Mathematical Meaning

Function represents:

- inputs and outputs
- mappings
- dependable rules
- relationships between quantities
- patterns of transformation

A function may send different inputs to the same output, but one input cannot
produce two different outputs under the same rule.

## Story Role

Function designs roads, bridges, machines, and pathways throughout Numeria.

Derivative often studies Function's paths to understand how their outputs
change when the inputs change.

## Canon Rule

Function's mappings must remain mathematically valid.

Every accepted input must connect to exactly one output.
