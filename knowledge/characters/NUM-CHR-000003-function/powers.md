# Function — Powers

## Path Mapping

Function connects each accepted input to exactly one output.

His paths may appear as:

- glowing roads
- bridges
- portals
- machines
- moving diagrams

## Rule Builder

Function can create a dependable rule that transforms an input into an output.

## Pattern Vision

Function can detect when different examples follow the same relationship.

## Limitations

- He cannot assign two outputs to the same input under one rule.
- He cannot work reliably when the domain is undefined.
- His powers do not explain how quickly an output changes; that belongs to Derivative.
