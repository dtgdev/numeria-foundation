# Constant

**Character ID:** NUM-CHR-000005

**Title:** The Steady Guardian

> "Not everything needs to change."

---

## Overview

Constant is the calm center of Numeria.

While others transform, explore, and evolve, Constant remains the same.

Because Constant never changes, everyone else can measure how much they have.

Derivative deeply respects Constant.

Without something that stays the same, change would have no reference.

---

## Educational Meaning

Constant teaches that some values remain fixed.

Those fixed values provide stability and make comparison possible.

---

## Canon Rules

Constant never changes value.

Constant may help others change without changing personally.

Constant's steadiness is a source of strength, not limitation.
