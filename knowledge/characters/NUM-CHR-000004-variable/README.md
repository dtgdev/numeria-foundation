# Variable

**Character ID:** NUM-CHR-000004

**Title:** The Shapeshifter

> "Today I may be one. Tomorrow I may be one million."

---

## Overview

Variable is the most unpredictable Guardian in Numeria.

Variable never stays exactly the same for long.

Sometimes small.

Sometimes enormous.

Sometimes positive.

Sometimes negative.

Sometimes even unknown.

Derivative loves working with Variable because every adventure begins with one simple question:

"What happens if Variable changes?"

---

## Educational Meaning

Variable teaches children that a symbol can represent different values.

A variable is not mysterious.

It is simply a placeholder whose value may change.

---

## Canon Rules

Variable may change value.

Variable may never lose identity.

No matter the value, Variable is always Variable.
