# Numeria Regions

Regions are major geographical or conceptual territories inside Numeria.

Each region must:

- Have a permanent canonical ID.
- Belong to the Numeria world.
- Represent a coherent mathematical environment.
- Contain locations, landmarks, institutions, artifacts, or stories.
- Support mathematically accurate learning experiences.

## ID Convention

Example: `NUM-REG-000001`

## Initial Region Candidates

- Kingdom of Algebra
- Forest of Functions
- Valley of Change
- Ocean of Integrals
- Geometry Gardens
- Probability Isles
- Infinity Mountains
