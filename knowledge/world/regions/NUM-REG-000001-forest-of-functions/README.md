# Forest of Functions

**Region ID:** `NUM-REG-000001`

The Forest of Functions is the first canonical region inside Numeria.

It is a living mathematical environment where functions transform
inputs into outputs and connected functions form larger pathways of change.

## Initial Story Connections

- Detective Derivative explores rates of change here.
- Captain Chain Rule follows linked transformations through the forest.
- Delta reveals how small changes travel through functions.
- The Ancient Function Tree records every transformation.
