# Numeria Character Guidelines

**Version:** 1.0 — Genesis

---

# Purpose

Characters are the heart of Numeria.

They transform abstract mathematical concepts into memorable personalities that learners can understand, relate to, and remember.

Every Numeria character exists for one primary purpose:

To make mathematics easier to understand.

---

# Philosophy

Children remember people more easily than symbols.

Instead of memorizing notation...

Learners meet characters.

Instead of memorizing formulas...

Learners experience adventures.

Every character should strengthen mathematical understanding.

---

# Character Design Principles

Every character should be:

- Mathematically Accurate
- Educational
- Memorable
- Visually Distinct
- Emotionally Relatable
- Consistent
- Timeless

No character exists only for entertainment.

Every character teaches.

---

# Character Template

Every canonical character should define:

## Identity

- Name
- Display Name
- Symbol
- Mathematical Concept
- Category
- First Appearance

---

## Personality

Describe:

- Core personality
- Strengths
- Weaknesses
- Fears
- Motivation
- Habits

Personality should naturally reflect the mathematics.

---

## Educational Purpose

Answer:

"What mathematical idea becomes easier because this character exists?"

If there is no clear answer, redesign the character.

---

## Story Role

Examples:

- Hero
- Mentor
- Explorer
- Inventor
- Scholar
- Guardian
- Rival
- Guide

Roles may evolve across stories.

---

## Visual Identity

Every character should have:

- Signature colors
- Clothing
- Icon
- Silhouette
- Symbol
- Expressions

Learners should recognize characters instantly.

---

## Mathematical Meaning

Describe:

- Definition
- Intuition
- Common uses
- Real-world examples
- Common misconceptions

This section keeps the character grounded in mathematics.

---

## Relationships

Every character should have meaningful relationships.

Examples:

- Mentor
- Student
- Friend
- Rival
- Family
- Team
- Opponent

Relationships should reinforce mathematical concepts.

---

## Growth

Characters should learn and develop.

Growth may reveal deeper mathematical ideas.

Growth must never contradict canonical mathematics.

---

# Naming Principles

Names should be:

- Easy to remember
- Easy to pronounce
- Educational
- Playful when appropriate

Avoid names that obscure the underlying concept.

---

# Humor

Humor is encouraged.

Confusion is not.

Jokes should reinforce learning rather than distract from it.

---

# Visual Consistency

Characters should share a recognizable artistic style while maintaining individual identities.

Across books, games, Studio, and animation, the same character should always be recognizable.

---

# Canon

There is only one canonical definition for each character.

Stories may deepen a character.

They may never contradict the mathematics that character represents.

---

# AI Usage

AI should understand every character through structured metadata, including:

- Personality
- Educational purpose
- Mathematical meaning
- Relationships
- Story history

This enables consistent storytelling and tutoring across the Numeria ecosystem.

---

# Closing

Great characters do more than entertain.

They create emotional connections that make mathematics unforgettable.

Every Numeria character should leave learners saying:

"I'll never forget that concept."
