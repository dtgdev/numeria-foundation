# Numeria Universe Explorer

**Version:** 1.0 — Genesis

---

# Purpose

The Universe Explorer is the primary gateway into the Numeria universe.

It allows learners, educators, creators, and developers to visually explore the relationships between mathematical concepts, characters, stories, worlds, artifacts, symbols, lessons, and educational pathways.

The Universe Explorer is not simply a graph.

It is a living map of mathematical knowledge.

---

# Vision

Imagine opening Numeria for the very first time.

Instead of opening a textbook...

You open an entire universe.

Characters move.

Worlds connect.

Ideas grow.

Relationships become visible.

Mathematics becomes something you can explore.

---

# Guiding Principles

The Universe Explorer should feel:

- Alive
- Beautiful

- Discoverable

# Core Experience

Users should be able to:

- Explore the universe visually
- Search anything

- Ask AI questions

# Entity Inspector

Selecting an Entity opens an inspector showing:

- Name
- Description
- Category
- Mathematical Meaning
- Educational Purpose
- Story Role
- Relationships
- First Appearance
- Connected Stories
- Related Lessons
- Related Symbols
- AI Summary

The inspector is the canonical profile for every entity.

---

# Graph Navigation

Users should be able to:

- Pan
- Zoom
- Center
- Expand
- Collapse
- Filter
- Highlight
- Bookmark
- Search
- Compare entities

Navigation should always remain smooth and intuitive.

---

# Search

Search should understand natural language.

Examples:

"Derivative"

"Show everything about change."

"Friends of Delta."

"What teaches Integrals?"

"Stories about Limits."

Search should be powered by both the Knowledge Graph and AI.

---

# AI Companion

An integrated AI guide should answer questions while users explore.

Examples:

"Explain why these concepts are connected."

"What should I learn next?"

"Why is Delta important?"

"Show me beginner-friendly stories."

The AI should always reference the canonical knowledge graph.

---

# Visual Language

Entity Categories should have consistent:

- Colors
- Icons
- Shapes
- Labels

Relationship types should have:

- Colors
- Line styles
- Animations
- Direction indicators

Visual consistency improves recognition.

---

# Educational Mode

Teachers should be able to:

- Hide advanced concepts
- Reveal concepts gradually
- Create guided tours
- Share custom learning paths
- Highlight important relationships

The Universe Explorer becomes an interactive classroom.

---

# Creator Mode

Creators should be able to:

- Add entities
- Edit relationships
- Build stories
- Connect curriculum
- Review canonical consistency

The Explorer is both a viewer and a creation tool.

# Accessibility

- Keyboard navigation

- High contrast themes

- Touch devices

Learning should be accessible to everyone.
