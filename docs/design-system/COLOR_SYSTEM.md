# Numeria Color System

**Version:** 1.0 — Genesis

---

# Purpose

The Numeria Color System establishes a consistent visual language across every Numeria experience.

Colors are not chosen only because they look good.

Each color communicates meaning.

Every application, book, animation, illustration, and interface should use the same semantic color system.

---

# Design Philosophy

Color should help learners think.

It should communicate structure before text.

It should guide attention without distraction.

The same mathematical ideas should always be represented using the same colors whenever possible.

Consistency builds recognition.

Recognition builds understanding.

---

# Primary Palette

## Numeria Blue

Represents knowledge, discovery, trust, and exploration.

Used for:

- Primary actions
- Links
- Navigation
- General mathematics

---

## Infinity Purple

Represents imagination, creativity, mystery, and storytelling.

Used for:

- Story Builder
- Creative Studio
- AI experiences
- Narrative content

---

## Growth Green

Represents learning, progress, success, and understanding.

Used for:

- Correct answers
- Completed lessons
- Growth indicators
- Positive feedback

---

## Energy Orange

Represents curiosity, experimentation, and exploration.

Used for:

- Interactive elements
- Discovery
- Challenges
- Recommendations

---

## Warning Amber

Represents caution without fear.

Used for:

- Validation
- Missing information
- Suggested corrections

Warnings should educate rather than intimidate.

---

## Error Crimson

Represents genuine problems requiring attention.

Errors should always explain:

- What happened
- Why it happened
- How to recover

Fear should never be part of the experience.

---

# Neutral Palette

Neutral colors provide balance.

Use them for:

- Backgrounds
- Borders
- Panels
- Secondary text
- Documentation

Content should always remain the visual focus.

---

# Character Colors

Major characters may have signature colors.

Examples:

Derivative

- Silver
- Blue

Delta

- Silver
- Purple

Infinity

- Deep Violet

Integral

- Emerald

Limit

- Midnight Blue

These identities should remain consistent across books, games, Studio, and animation.

---

# Relationship Colors

Relationships should also communicate meaning.

Examples:

Inheritance

Blue

Friendship

Green

Conflict

Red

Discovery

Orange

Transformation

Purple

Mentorship

Gold

The same relationship should always use the same color throughout Numeria.

---

# Accessibility

Color alone should never communicate meaning.

Every color-coded element should also include:

- Labels
- Icons
- Patterns
- Shapes

The system should remain usable for every learner.

---

# Future Expansion

Additional palettes may be introduced for:

- Themes
- Seasonal artwork
- Classroom modes
- Accessibility modes
- Dark mode
- Print editions

These should extend the system rather than replace it.

---

# Closing

Color is part of the language of Numeria.

When learners see a color, they should immediately understand its purpose.

Consistency transforms colors into meaning.
